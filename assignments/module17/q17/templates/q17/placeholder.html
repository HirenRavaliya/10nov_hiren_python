Implementation Pending

Twitter API v2 now requires a paid subscription (Basic plan: $100/month) for timeline access.

Required .env Key:
TWITTER_BEARER_TOKEN=your_bearer_token_here
API Endpoint to Use:
GET https://api.twitter.com/2/users/:id/tweets?max_results=5
Authorization: Bearer {BEARER_TOKEN}
Implementation Code:
import requests
headers = {"Authorization": f"Bearer {settings.TWITTER_BEARER_TOKEN}"}
# Get user ID from username
user_resp = requests.get(
    f"https://api.twitter.com/2/users/by/username/{{username}}",
    headers=headers
)
user_id = user_resp.json()['data']['id']
# Get tweets
tweets_resp = requests.get(
    f"https://api.twitter.com/2/users/{{user_id}}/tweets?max_results=5",
    headers=headers
)
