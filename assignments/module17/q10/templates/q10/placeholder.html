This app requires live API credentials:
Google OAuth2 Client ID/Secret and Twilio Account SID/Token.

Required .env Keys:
TWILIO_ACCOUNT_SID=your_sid
TWILIO_AUTH_TOKEN=your_token
TWILIO_PHONE_NUMBER=+1234567890
# Google OAuth2 — configure in Google Cloud Console
Packages:
pip install django-allauth twilio
Implementation Steps:
  1. Add allauth and allauth.socialaccount.providers.google to INSTALLED_APPS
  2. Configure Google OAuth2 in Django Admin → Social Applications
  3. On login, generate OTP and send via Twilio SMS
  4. Verify OTP before granting access
